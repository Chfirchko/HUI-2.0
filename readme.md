# HUI - Horoshiy User Interface


## Сборка через CMake


```bash
cd build
cmake -DCMAKE_EXPORT_COMPILE_COMMANDS=ON ..
make -j12
```

## Структура папок

- `build/` — папка для сборочных файлов 
- `CMakeLists.txt` — файл конфигурации сборки CMake
- `include/` — заголовочные файлы
  - `common/` — общие хедеры, используемые в разных частях проекта
  - `data/` — хедеры для обработки и сохранения данных
    - `PacketProcessor.h` — объявление класса для обработки пакетов
  - `gui/` — хедеры для графического интерфейса (пока пусто)
  - `serial/` — хедеры для работы с последовательным портом
    - `SerialPort.h` — объявление класса для работы с serial портом
- `readme.md` — файл с описанием проекта
- `src/` — исходники
  - `data/` — исходники для обработки и сохранения данных
    - `PacketProcessor.cpp` — реализация обработки пакетов
  - `gui/` — исходники для графического интерфейса (пока пусто)
  - `serial/` — исходники для работы с serial портом
    - `SerialPort.cpp` — реализация работы с serial портом
  - `main.cpp` — точка входа в программу

# HUI - Horoshiy User Interface


## Сборка через CMake


```bash
cd build
cmake -DCMAKE_EXPORT_COMPILE_COMMANDS=ON ..
make -j12
```

## структура папок
├── include/                  # Заголовочные файлы (.h, .hpp)
│   ├── gui/                  # Хеддеры для GUI
│   ├── serial/               # Хеддеры для работы с serial портом
│   ├── data/                 # Хеддеры для обработки и сохранения данных
│   └── common/               # Общие хеддеры, если есть
│
├── src/                      # Исходники (.cpp)
│   ├── gui/                  # Исходники для GUI
│   ├── serial/               # Исходники для serial порта
│   ├── data/                 # Исходники для обработки и сохранения данных
│   └── main.cpp              # Точка входа (если один файл)
│
├── resources/                # Ресурсы (иконки, шрифты и прочее для GUI)
│
├── build/                    # Сборочные файлы (обычно игнорируется в гите)
│
└── CMakeLists.txt            # Конфигурация сборки (если используешь CMake)
